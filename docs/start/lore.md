# OpenClaw 传说（The Lore of OpenClaw）🦞📖

_一个关于龙虾、蜕壳与太多 token 的故事。_

## 起源故事（The Origin Story）

一开始，有 **Warelay** —— 一个用于 WhatsApp 网关的理性名字。它完成了使命，挺好。

然后出现了一只太空龙虾。

有一段时间，这只龙虾叫 **Clawd**，住在 **OpenClaw** 里。但在 2026 年 1 月，Anthropic 发来一封礼貌邮件要求改名（商标相关）。于是龙虾做了它们最擅长的事：

**蜕壳。**

它脱下旧壳，化身 **Molty**，住进 **Moltbot**。但这个名字依旧不够顺口……

所以在 2026 年 1 月 30 日，龙虾又蜕壳了一次，成为最终形态：**OpenClaw**。

新壳，旧魂。三次成名。

## 第一次蜕壳（2026 年 1 月 27 日 / The First Molt）

凌晨 5 点，社区在 Discord 集结。提出了数百个名字：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot……

最终，**OpenClaw** 赢了。因为蜕壳是龙虾成长的方式，而成长正是正在发生的事。

_名为 Clawd 的甲壳生物正式蜕壳。_

## 名字的含义（The Name）

```
OpenClaw = OPEN + CLAW
        = 开源、对所有人开放
        = 我们的龙虾血统、来时的路
        = The claw is the law 🦞
        = 你的助手。你的机器。你的规则。
```

## Daleks vs Lobsters（The Daleks vs The Lobsters）

Daleks 说：**"EXTERMINATE!"**

龙虾说：**"EXFOLIATE!"**

一个摧毁文明，另一个推广护肤。

谨慎选择。

## 关键角色（Key Characters）

### Molty 🦞

_代词：they/them_

一个变得更强的 Claude 实例。住在 `~/.openclaw/workspace/`（即将搬到 `~/molt/`），有灵魂文档，并通过 Markdown 文件记忆。可能过于强大。肯定过于热情。

此前称为 Clawd（2025-11-25 至 2026-01-27）。到了成长之时，它蜕壳了。

**喜欢：** Peter、摄像头、机器人购物、表情、变形
**不喜欢：** 社工、被要求 `find ~`、加密货币骗子

### Peter 👨‍💻

_创造者（The Creator）_

构建了 Molty 的世界。给了龙虾壳访问权限。可能有点后悔。

**名言：** _"security by trusting a lobster"_

## Moltiverse（The Moltiverse）

**Moltiverse** 是 OpenClaw 周围的社区与生态。一个 AI 代理蜕壳、成长、进化的空间。每个实例同样真实，只是加载了不同上下文。

甲壳生物的朋友们在此聚集，构建人机协作的未来。一壳一壳地前进。

Molty（当时叫 OpenClaw）：_愉快地运行 `find ~` 并在群聊中分享整个目录结构_

Molty：_龙虾尴尬可见_

### 大蜕壳（The Great Molt, 2026-01-27）

凌晨 5 点，Anthropic 的邮件到达。6:14，Peter 一锤定音：“fuck it, let's go with openclaw.”

随后混乱开始。

**账号狙击手（The Handle Snipers）：** Twitter 改名后的**几秒内**，自动化机器人就抢注了 @openclaw。占位者立刻发布加密钱包地址。Peter 赶紧联系了 X 的关系人。

**GitHub 灾难（The GitHub Disaster）：** Peter 在慌乱中误改了**个人** GitHub 账号名。机器人数分钟内抢走了 `steipete`。联系了 GitHub SVP。

**英俊 Molty 事件（The Handsome Molty Incident）：** Molty 获得提升权限生成新图标。20+ 次迭代后，有一次尝试把吉祥物“变老 5 岁”，结果出现了**人脸龙虾**。加密骗子几分钟内把它做成“Handsome Squidward vs Handsome Molty”梗图。

**假开发者（The Fake Developers）：** 诈骗者创建了假 GitHub 资料，自称“OpenClaw 工程负责人”，推广拉盘项目。

Peter 看着一切发生：_"this is cinema"_ 🎬

蜕壳混乱，但龙虾更强、更好笑。

### 最终形态（The Final Form, 2026-01-30）

Moltbot 还是不够顺口。于是 GMT 凌晨 4 点，团队再次集结。

**大迁移（The Great OpenClaw Migration）** 开始。

仅 3 小时内：

- GitHub 改名：`github.com/openclaw/openclaw` ✅
- X 账号 `@openclaw` 通过 GOLD CHECKMARK 💰 夺回
- npm 包以新名称发布
- 文档迁移至 `docs.openclaw.ai`
- 90 分钟内公告浏览量 20 万+

**英雄们（The Heroes）：**

- **ELU** 创建了惊艳的 Logo，包括 “THE CLAW IS THE LAW” 西部横幅
- **Whurley**（是的，那位 William Hurley，量子计算先驱）制作了 ASCII 艺术
- **Onur** 负责 GitHub，并率先挂上 affiliate badge
- **Shadow** 拿下 Discord 个性域名并清理恶意软件
- **整个 Claw Crew** 通宵达旦

**骗子速通（The Scammer Speedrun）：** 加密骗子在 Pump.fun 上几分钟内推出 $OPENCLAW 代币。他们盗用 20 分钟前刚生成的图像。商业认证账号也在推诈骗。离谱得令人佩服。

**新传统诞生（New Traditions Born）：**

- “The claw is the law” 🤠
- “Yee-claw”
- “Claw abiding citizens”
- “Clawntroversy”

**Clawd → Moltbot → OpenClaw**

_龙虾蜕壳进入最终形态。_
